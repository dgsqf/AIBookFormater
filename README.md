# AIBookFormater

A python CLI to create a book from a text


### Prerequisites



- Python 3.9



### Installing



Clone the repo

```
$ git clone https://github.com/dgsqf/AIBookFormater.git
```

cd into it

```
$ cd AIBookFormater
```

use it

```
$ python --inp (path to your input file) --out (output file)
```



for input format refer to the wiki
